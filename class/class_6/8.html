  4. Answer the following question.
    1. What is a chart in MS Excel?
      Ans. A chart is a pictorial representation of data. It allows us to illustrate the data graphically
    2. Write a few words about column chart.
      Ans. Column chart displays the data in form of vertical columns. It typically displays catrgories along the horizontal axis and values along the vertical axis.
    3. Write the steps to move a chart on the worksheet.
      Ans. Steps to move a chart on the worksheet
      1. Point mouse cursor anywere on the chart area.
      ( It converts into handle ).
      2. Click and drag the chart to new location.
    4. Write the steps to change chart style in a worksheet.
      Ans. Steps to change chart style in a worksheet
      1. select the chart on the worksheet.
      ( The Chart tool having Design and Format tabs appear on Ribbon. )
      2. Click on Design tab.
      3. Click On the drop-down button of Chart Styles.
      4. Select the required style from the menu.
    5. Write the steps to create a chart on a worksheet.
      Ans. Steps to creat a chart on a worksheet
      1. select the ( cell ) range we want to chart including column title.
      2. On Insert tab, in Chart group, Click on desired chart type.
      3. Select the required chart sub-type from the drop-down list.
    6. Write the steps to insert a picture in a worksheet in Excel.
      Ans. Steps to insert a picture in a worksheet in Excel.
      1. Select any cell.
      2. On Insert tab, in Illustrations group, Click on Pictures tool.
      ( The Insert Picture dialog box appears. )
      3. Locate the required image and select it.
      4. Click on Insert button.
